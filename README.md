# 中興大學土雞品系與農場導入指南

## 總覽
- **4 個育種品系**（B公系、S公系、L2母系、D母系）+ **14 個保種品系**
- **44 年選育歷史**（1982-2026），李淵百教授創立
- **全球唯二**完整體系：法國 + 台灣（AgroParisTech 1999 年確認）
- **3,500+ 隻**保種族群（烏日溪心壩畜產場，12 棟雞舍）

## 農場導入四大重點品系

### 1. 中興紅羽土雞（中興紅羽1982）
- 44 年選育，李淵百教授創立，L2 母系 × B 公系
- 公 2.5-3.0 kg / 母 1.8-2.2 kg，飼養 14-18 週，FCR 2.8-3.2:1
- IMP 肌苷酸 2.5-3.5 mg/g（高白肉雞 133%）
- **農場適合度：★★★★★** — 已導入屏東大武、台中谷關、南投奧萬大

### 2. 興大土雞101品系
- 2012 年命名，全台第一個大學蛋雞品系，陳志峰教授團隊
- 年產蛋 200-220 顆，蛋重 48-52g，蛋黃比例 32-35%
- 每隻種母雞年產可售雛雞 150-170 隻
- **農場適合度：★★★★☆**

### 3. 金門土雞
- 27 年 24 世代保種，金門海島隔離演化
- 公 1.6-2.0 kg / 母 1.2-1.5 kg（最小體型）
- 基礎代謝率低 8-12%，高溫採食下降僅 15-20%
- **農場適合度：★★★★☆**

### 4. 絲羽烏骨雞（Silkie）
- 1,500+ 年歷史，《本草綱目》藥雞
- 總胺基酸 18,832 mg/100g（全台最高），BCAA 3,287 mg/100g
- 低脂肪 4-6%，興大已量產烏骨雞滴雞精
- **農場適合度：★★★☆☆**

## 14 個保種品系
台灣地方種原：花東土雞、信義土雞、竹崎土雞、金門土雞、峨嵋土雞、部落雞（萬大部落）
國際引入種原：文昌雞（中國）、鬥雞、名古屋雞（日本）、絲羽烏骨雞（中國）、法國土雞
