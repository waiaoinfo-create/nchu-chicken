# 中興大學土雞品系與農場導入指南

## 總覽
- **4 個育種品系**（持續選育中）+ **14 個保種品系**（種原保存）
- **44 年選育歷史**（1982-2026），黃糪苳教授創立
- **全球唯二**完整體系：法國 + 台灣（AgroParisTech 1999 年確認）
- **3,500+ 隻**保種族群

## 4 個育種品系

### 1. 紅羽土雞（1982 基礎品系）
- 44 年選育，黃糪苳教授創立
- 公 2.5-3.0 kg / 母 1.8-2.2 kg，飼養 14-18 週，FCR 2.8-3.2:1
- IMP 肌苷酸 2.5-3.5 mg/g（高白肉雞 133%），肌纖維 40-55 μm（細 30%）
- 膠原蛋白 2.5-3.5%（高 150%）
- **農場適合度：★★★★★** — 已導入屏東大武、台中谷關、南投奧萬大、台東阿拉巴灣

### 2. 中興大學 101 土雞
- 2012 年命名，全台第一個大學蛋雞品系，陳志峰教授團隊
- 年產蛋 200-220 顆，高峰產蛋率 85-90%，蛋重 48-52g，蛋黃比例 32-35%
- 哈氏單位 75-85（新鮮度極佳）
- **農場適合度：★★★★☆** — 蛋用為主，適合撿蛋體驗 + 蛋品販售

### 3. 長榮土雞（黑羽土雞）
- 27 年 24 世代，耐熱抗病最強
- 公 1.6-2.0 kg / 母 1.2-1.5 kg（最小型）
- 基礎代謝率低 8-12%，高溫存活率 85-90%
- **農場適合度：★★★★☆** — 抗病最強、成本最低，適合林下放養

### 4. 烏骨雞（中國竹絲雞／Silkie）
- 1,500+ 年歷史，《本草綱目》藥雞
- 總胺基酸 18,832 mg/100g（全台最高），BCAA 3,287 mg/100g
- 低脂肪 4-6%，單元不飽和脂肪酸 60-65%
- 10 大特徵：桑椹冠、絲羽、鬍鬚、藍綠耳、纓冠、腳毛、黑骨、黑肉、黑黏膜、黑趾
- **農場適合度：★★★☆☆** — 需搭配加工（雞精/藥膳），中興大學已量產烏骨雞精
